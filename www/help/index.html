What is it?

Do you use a Desktop launch bar on your computer? Maybe Alfred, Quicksilver or something similar? Well Storm Bar is just like that but built for the web.

Imagine a launch bar for the web that has a documented public JavaScript based API where extensions are simply URLs. This is Storm Bar.

Read more about Storm Bar…

How do I use it?

First you'll need to install some Bolts. Bolts are extensions that allow you to add new commands to your Storm Bar.

Storm Bar ships with 2 Bolts installed by default. These are 'help' and 'bolt'. You've already seen what the 'help' Bolt does.

The 'bolt' Bolt is a very simple package manager allows you to list, install, update and remove Bolts from your local machine.

First lets try installing a bolt that allows you to search Google. Open Storm Bar in another window and try type 'bolt install google' then hit enter.

Now you should be able to type 'google funny cat' and get some search results back. Congratulations, you installed and used your first Bolt!

Where can I find more Bolts?

Storm Bar is a very young project so there aren't too many Bolts around yet. There is a community repository of useful bolts on github. When you type in a bolt name like 'bolt install google' it is installed from here.

Alternatively you can write your own custom bolts and install them with a URL (gists make a great home for custom Bolts) 'bolt install http://mydomain.com/mybolt.js'

For more help on writing your own bolts see the Bolt API guide.
